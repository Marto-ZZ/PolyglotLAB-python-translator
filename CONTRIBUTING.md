# Contribuir a PolyglotLab

¡Gracias por tu interés en **PolyglotLab**! Este es un proyecto personal que forma parte de mi portfolio profesional. 

## 🚀 ¿Cómo puedes ayudar?

Aunque no busco colaboradores para desarrollar nuevas funcionalidades directamente en el código base, valoro muchísimo tu feedback en las siguientes áreas:

* **Reporte de Errores (Bugs):** Si encuentras algo que no funciona como debería, por favor abre un *Issue*.
* **Sugerencias de Mejora:** ¿Crees que una función podría ser más eficiente o el diseño más limpio? ¡Me encantaría leer tus ideas en la sección de *Issues*!
* **Feedback de Código:** Si eres un desarrollador senior y ves algo que podría optimizarse, tus consejos son bienvenidos.

## 🛠️ Proceso para enviar sugerencias

1.  **Abre un Issue:** Describe claramente tu sugerencia o el error encontrado.
2.  **Discusión:** Responderé lo antes posible para intercambiar ideas.
3.  **Pull Requests:** Por favor, **no envíes Pull Requests directamente** sin antes haber discutido el cambio en un Issue. Como este es un proyecto de portfolio, la mayoría de los cambios los implementaré yo mismo para mantener la integridad de mi trabajo, dándote el crédito correspondiente por el hallazgo.

## ⚠️ Nota sobre el uso del código

Este proyecto está bajo una licencia de **Todos los Derechos Reservados**. Se permite la inspección del código con fines de revisión técnica, pero **no se permite la copia total o parcial, ni la redistribución** del mismo para otros portfolios o proyectos comerciales.
