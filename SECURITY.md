## Reportar una vulnerabilidad

Si descubres una vulnerabilidad de seguridad en este proyecto, por favor **no abras un Issue público**. En su lugar, puedes contactarme directamente a través de mi perfil de GitHub o mi correo electrónico personal.

Agradezco mucho tu ayuda para mantener este proyecto seguro.
